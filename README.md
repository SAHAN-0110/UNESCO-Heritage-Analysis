# UNESCO World Heritage Analysis

This project analyzes UNESCO World Heritage Sites using Tableau.

## Dashboards
1. Global Distribution of Heritage Sites (Treemap + Top 10 Countries)
2. Risk Status & Regional Inscription Trends (Pie + Line Chart)

## Tools Used
- Tableau Public
- Python
- Flask

## Web Integration
The Tableau dashboard is integrated into a Flask web application.
